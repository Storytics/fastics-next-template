# Storytics NextJS Template

A boilerplate for NextJS with batteries included (TypeScript, Styled-Components, Eslint, Prettier, Jest).

### Stack

- TypeScript
- Prettier
- Eslint
- Styled-Components
- styled-system
- React-intl

### Install

#### with npm

```shell
npm install
```

#### with Yarn

```shell
yarn install
```

### Starting the development server

```shell
yarn dev
```
